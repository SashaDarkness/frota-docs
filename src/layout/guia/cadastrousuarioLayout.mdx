  <section>
    <p class="text-lg font-bold">COMO CRIAR UM PERFIL NO SISTEMA?</p>
    Para realizar cadastro de novos perfis o usuário deve acionar a opção <b>"Novo"</b> localizado no lado direito superior da tela <i>(conforme indicado na imagem acima)</i>.
    Em seguida, será direcionado para a tela de cadastro onde deverá preencher um formulário informando o Nome do Usuário, caso o usuário informado seja encontrado aparecerá
    no campo de listagem para ser selecionado. Após selecionar o usuário, o campo de login será preenchido automaticamente, a seguir o usuário deve informar o tipo de perfil que ele deve pertencer.

  <section>
    <p class="text-lg font-bold">EXCLUÇÃO DE PERFIL NO SISTEMA</p>
    Para realizar a exclusão de um perfil no sistema o usuário deve acionar a opção de <b>[Excluir](#ancora1)</b> <i>(como apresentado na imagem inicial)</i>. Em seguida, 
    o sistema exibe ao usuário a tela abaixo: 
    
    <div class="md:shrink-0">
      <div class="w-full h-[510px] overflow-auto touch-auto bg-transparent">

      </div>
    </div>

    ● Ao entrar na tela é exibido os dados do usuário, para dar procedimento à exclusão é necessário clicar em <b>"Confirmar"</b>. Caso ao contrário poderá retornar para tela inicial
    acionando a opção de <b>“Cancelar”</b>. Se a exclusão ocorrer com êxito aparecerá uma mensagem no canto superior da tela informando que os dados foram excluídos com sucesso

    ● Nesta mesma tela o sistema disponibiliza ao usuário a opção para: adicionar um novo perfil de usuário (conforme indicado na imagem acima).
